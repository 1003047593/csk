# 集群的生命周期

|状态|说明|
|--|--|
|待激活（inactive）|集群创建成功，但是集群中没有节点|
|初始化中（initial）|集群正在申请相应的云资源|
|运行中（running）|集群申请云资源成功|
|更新中（updating）|集群升级 Agent 版本|
|伸缩中（scaling）|变更集群的节点数量|
|创建失败（failed）|集群申请云资源失败|
|删除中（deleting）|集群删除中|
|删除失败（delete\_failed）|集群删除失败|
|已删除（deleted，该状态用户不可见）|集群删除成功|
